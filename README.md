# 豆包下载器

> 豆包 AI 生成的视频和图片 — **一键无水印下载**，无限制次数

## ✨ 功能

- 🖼️ **图片下载** — 捕获豆包 AI 生成的图片，直接保存无水印版本
- 🎬 **视频下载** — 捕获豆包 AI 生成的视频，直接保存无水印版本
- ♾️ **无限制** — 不限制下载次数，随意使用
- 🚀 **一键操作** — 页面内自动捕获，无需手动复制链接

## 📦 安装

1. 打开 Chrome 浏览器，进入扩展管理页面 `chrome://extensions/`
2. 开启右上角 **开发者模式**
3. 点击 **加载已解压的扩展程序**
4. 选择本项目所在的文件夹

## 🛠️ 技术说明

- **Manifest V3** — 使用 Chrome 最新扩展规范
- **Content Script** — 注入豆包页面，拦截媒体数据
- **Service Worker** — 后台处理下载任务
- **Popup UI** — 显示激活状态和统计信息

> 扩展加载后需要输入激活码才能使用。
